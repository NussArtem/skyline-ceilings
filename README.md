# SkyLine Ceilings - Website

Сайт компании SkyLine Ceilings - премиальные тканевые потолки Descor и HEYtex.

## Структура проекта

```
skyline-ceilings/
├── index.html          # Главная страница
├── pages/              # Дополнительные страницы
├── styles/             # CSS стили
│   └── main.css        # Основные стили в строгом минималистичном стиле
├── scripts/            # JavaScript файлы
│   └── main.js         # Основная логика (галерея, формы, меню)
├── images/             # Изображения (добавьте свои фото)
└── package.json        # Зависимости проекта
```

## Особенности

- **Строгий минималистичный дизайн** - чистые линии, нейтральные цвета, профессиональный вид
- **Адаптивная верстка** - сайт корректно отображается на всех устройствах
- **Интерактивная галерея** - увеличение фотографий с навигацией
- **Форма обратной связи** - заявка на замер
- **Мультиязычность** - поддержка RU, EN, ES (базовая структура)
- **Современный код** - чистый HTML5, CSS3, Vanilla JavaScript

## Установка и запуск

1. Установите зависимости:
```bash
npm install
```

2. Для локального просмотра используйте любой статический сервер, например:

```bash
# С помощью Python
python3 -m http.server 8000

# Или с помощью Node.js (npx)
npx serve

# Или с помощью PHP
php -S localhost:8000
```

3. Откройте браузер и перейдите по адресу `http://localhost:8000`

## Форматирование кода

Проект использует Prettier для форматирования кода:

```bash
# Форматировать все файлы
npm run format

# Проверить форматирование
npm run format:check
```

## Настройка

### Изменение цветов
Отредактируйте CSS переменные в `styles/main.css`:

```css
:root {
    --color-primary: #1a1a1a;
    --color-secondary: #2d2d2d;
    --color-accent: #000000;
    /* ... */
}
```

### Добавление контента
Все тексты находятся в HTML файлах. Для изменения контента отредактируйте соответствующие HTML файлы.
